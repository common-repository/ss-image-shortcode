== Description ==

With SS Image Shortcode you can add animated images with captions everywhere. Also, you can create a gallery, just adding images with the shortcode manager at the same page.

**How to use**

Create a new page/post. Inside the Default editor you will find a new button that have a drawing representing an image, click it to adjust settings of each image, and that's it.

**List of available Options:**

* Image Sizes (one half, one third, one fourth, full size)
* Title Field
* Caption Field
* Overlay Background Color
* Text Color

Live Demo 
http://miguras.com/plugins/?page_id=4


== Installation ==

1. Upload the files to the `/wp-content/plugins/ss_image_shortcode/` directory or in your wordpress admin panel navigate to plugins->add new upload the installable zip file.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Go to a page, press the SS shortcode (this button was added inside the default wordpress editor and has a drawing representing an image) button and begin add styled images.
